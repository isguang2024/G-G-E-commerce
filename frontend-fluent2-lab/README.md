# Fluent 2 React Lab

一个独立的 Fluent UI React v9 测试页面集合，用来验证 Fluent 2 风格、后台高频页面骨架，以及 `Fluent 2 Web` 与 `Teams` 两种布局模式如何共存。

## 目标

- 不接现有 Vue 工程
- 不接现有后端接口
- 先验证页面壳层、组件组合、布局节奏和视觉方向
- 用一套共享 Fluent 基座承载 `Fluent 2 Web` 与 `Teams` 两种页面语法
- 通过多种场景骨架扩展更多工作台、规范页和协作页，而不是只换文案复制整页

## 当前实验能力

- 顶层支持按 `共享基座`、`基础控件集`、`Fluent 2 Web`、`Teams` 分组切换页面
- 顶层支持亮色 / 暗色主题快速预览
- 已抽出共享实验室基元，供频道页、消息线程页、审批工作台和规范工作区复用
- 已按场景拆成指挥台、工作台、目录页、规范页、评审页、频道页、线程页、公告页等多种骨架
- 已新增基础图标总览页，支持浏览 `@fluentui/react-icons` 的基础图标 ID 并点击复制
- 已新增三张组合模式页，把基础组件真实收口成“导航指挥面”“表单反馈面”“内容协作面”
