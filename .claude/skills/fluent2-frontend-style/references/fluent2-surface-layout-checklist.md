# Fluent 2 布局与表层检查清单

## 应用壳

- 顶部栏是否只放高价值全局元素
- 导航区是否稳定且不拥堵
- 主工作区是否最突出
- 次级面板是否承担辅助任务

## 表层与材质

- 是否只在必要时使用材质和阴影
- 阴影是否用于层级，而不是装饰
- 是否避免了大面积花哨渐变与厚重玻璃感
- 表层变化是否有明确交互原因

## 排版

- 标题、正文、说明、状态的字号层级是否清楚
- 是否优先左对齐，保证扫读效率
- 文本行长是否可读
- 缩放与窄屏下是否仍稳定

## 颜色与状态

- 主色是否只承担少量高价值动作
- 状态色是否克制且一致
- 是否存在为装饰而滥用颜色的情况
- 信息是否不仅依赖颜色区分

## 工作台页面

- 首屏是否能看出主任务
- 主操作是否贴近主对象
- 列表、详情、属性和消息层是否分工明确
- 是否避免了卡片瀑布流和营销式视觉
