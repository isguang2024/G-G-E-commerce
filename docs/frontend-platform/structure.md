## 说明

`frontend-platform/` 是基于 Vben Admin 模板的前端工作空间，采用 monorepo 结构。具体结构以实际目录为准，当前尚未沉淀到正式项目说明。

## 顶层目录（当前）

- `apps/`：应用实例
- `packages/`：内部共享包
- `internal/`：内部工具与脚手架
- `playground/`：开发调试场
- `scripts/`：构建/运维脚本
- `docs/`：原项目文档（非真相源）
