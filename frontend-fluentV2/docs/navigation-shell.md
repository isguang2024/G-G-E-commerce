# 导航与壳层规则

## 壳层职责

- 顶部栏：品牌、侧栏控制、菜单搜索、主题切换、用户区
- 左侧导航：模块入口、一级/二级层级、折叠/展开、当前高亮
- 标签栏：展示当前已打开页面，负责页签切换与关闭
- 内容区：面包屑、标题、副标题、页面容器

## 路由机制

- 使用 `HashRouter`
- 业务组件不直接拼 `#/`
- `route-registry.tsx` 统一维护路由定义与页面映射
- 已定义未实现的页面统一进入 `MigrationPlaceholderPage`
- 未匹配路径进入 `NotFoundPage`

## 菜单空间

当前使用本地 mock：

- `default`
- `platform-governance`
- `team-collaboration`

切换空间后：

- 更新壳层上下文
- 过滤侧边导航
- 跳转到该空间的默认入口

## 导航组织

首期保留当前 Vue 主线的顶层信息架构：

- Welcome
- Workspace
- Team
- Message
- System

除 `Welcome / Workspace / System / System Menu` 外，其余路由均使用统一占位页。

## 标签栏规则

- 标签栏位于顶部栏下方、页面内容上方，属于壳层而不是页面内部组件
- 访问一个已注册路由时自动打开对应标签；相同路径不重复开标签
- 当前激活页签与当前路由保持同步
- 关闭当前页签时自动跳转到相邻已打开页签；仅剩一个页签时不提供关闭动作
- 支持固定标签；固定标签会保持在左侧，且不会出现在普通关闭按钮流里
- 支持关闭其他标签
- 支持右键标签上下文菜单，可执行固定、关闭其他、关闭左侧、关闭右侧与关闭当前
- 支持桌面端拖动重排；固定标签与普通标签仍保持分区排序
- 支持标签持久化恢复；刷新页面或重新进入后台后，已打开标签会恢复
- 顶部栏左侧功能区提供“显示标签栏”开关；关闭后隐藏标签栏，并停止为后续路由访问自动开标签
- 标签拖动重排采用 `Pointer Events` 驱动的自定义拖拽层；拖动标签实体跨过目标标签中线后立即换位
- 标签换位时，其他受影响标签使用外层 wrapper 级让位动画，从当前屏幕位置平滑过渡到新位置
- 退出登录时清空当前会话的已打开页签
