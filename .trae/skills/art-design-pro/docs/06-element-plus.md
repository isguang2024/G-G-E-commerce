# 组件库 | Art Design Pro

来源：https://www.artd.pro/docs/zh/guide/essentials/element-plus.html

## 概述

项目组件库使用 Element Plus 提供强大支持，覆盖了 80% 的常用组件，同时系统内置了丰富的组件、模版，方便快速开发。

为了获得更好的视觉效果，本项目对 Element Plus 部分组件 UI 进行了调优。

**样式文件路径**：`src/assets/styles/el-ui.scss`

## Element Plus

基础 UI 组件库，提供：
- 按钮、表单、表格等基础组件
- 弹窗、消息提示等反馈组件
- 布局、容器等结构组件

官方文档：https://element-plus.org/

## 自定义样式

如果需要自定义 Element Plus 组件样式，修改以下文件：

```scss
// src/assets/styles/el-ui.scss
@forward 'element-plus/theme-chalk/src/common/var.scss' with (
  // 自定义主题色
  $colors: (
    'primary': (
      'base': #409EFF,
    ),
  ),
);
```

## 💡 最佳实践

1. **优先使用系统组件** - 系统组件已经过优化和测试
2. **参考官方文档** - Element Plus 官方文档提供完整 API
3. **统一样式** - 保持整个项目的视觉一致性
4. **按需导入** - 使用按需导入减少打包体积
